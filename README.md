# House Price Prediction using Linear Regression

This project builds a Linear Regression model to predict house prices using the California Housing dataset.

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Workflow
1. Data loading
2. Exploratory Data Analysis (EDA)
3. Train-test split
4. Linear Regression training
5. Model evaluation

## Dataset
California Housing Dataset (scikit-learn)
